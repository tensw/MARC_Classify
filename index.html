1단계 · ISBN 입력 사람

입력
13자리 ISBN
처리
입력값 유효성 확인 + 도서 매칭
출력
다음 단계로 진행 (도서 식별)

2단계 · 도서 기본 정보 호출 자동 수정 가능

입력
ISBN
처리
국립중앙도서관 API에서 서지 호출 + 책 지문 17필드 LLM 생성
출력
서지 7개 필드 + Fingerprint 17개 필드 (편집 가능)

3단계 · DDC 분류 매칭 자동 선택 가능

입력
책 지문 17필드
처리
SKKU DDC 사전 25,000여 항목과 의미 비교, 매칭 점수 산출
출력
추천 1개 + 대체 후보 2개 (점수와 근거 포함)

4단계 · MARC 자동 생성 자동 수정 가능

입력
서지 + 책 지문 + DDC
처리
SKKU MARC21 사전(361,353건 분석) 기반으로 18필드 생성, 의존관계·구두점 규칙 적용
출력
MARC 18필드 (필수 11 + 권장 7), 사용자 친화/Raw 토글 표시

5단계 · 최종 검수 사람 (필수)

입력
4단계 결과 전체
처리
자동 검증 7건 결과 + 검수자 책임 체크박스 확인
출력
검수자 승인 후 정식 등록, 대시보드 카운트 갱신
