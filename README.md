# Biblo: 학술정보관 MARC 자동 생성기 — 목업

시연용 정적 HTML 목업. `index.html`을 더블클릭하면 브라우저에서 바로 열립니다.

## 5단계 흐름
1. ISBN 입력 + 대시보드 + 완료 MARC 리스트
2. 도서 기본 정보 호출 + 책 지문 17필드
3. DDC 분류번호 자동 매칭
4. MARC 레코드 자동 생성
5. 최종 검수 (사람 책임 확인)

## 시연용 도서 5권
- 소년이 온다 / 한강
- 작별인사 / 김영하
- 사피엔스 / 유발 하라리
- 정재승의 과학콘서트 / 정재승
- 박태웅의 AI 강의 2026 / 박태웅

## 데이터 갈아끼우기
`data/*.js` 안의 `window.SAMPLE_BOOKS` 등 전역 객체를 직접 수정하면 됩니다.

## 데이터 파일 구조

| 파일 | 역할 | 갈아끼우기 단위 |
|------|------|-----------------|
| `data/sample-books.js` | 1단계 칩 5권 요약 (ISBN/제목/저자/출판사/표지) | `window.SAMPLE_BOOKS` 배열 항목 추가/교체 |
| `data/book-details.js` | 2단계 서지 7필드 + 책 지문 17필드 | `window.BOOK_DETAILS[isbn]` 객체 |
| `data/ddc-dict.js` | 3단계 추천 1 + 대체 2 후보 | `window.DDC_CANDIDATES[isbn]` 객체 |
| `data/marc-fields.js` | 4단계 MARC 18필드 (메인 14 + 접힘 4) | `window.MARC_RECORDS[isbn]` 객체 |

새 ISBN을 추가하려면 4개 파일 모두에 동일 ISBN 키로 항목을 추가해야 합니다.

## 시연 권장 시나리오

1. **소년이 온다** — 오리지널 한국 소설 기본 흐름 (수상주기 586 시연)
2. **사피엔스** — 번역서 분기 (041 $h, 246, 700, 900 모두 생성)
3. **과학콘서트** — 개정증보판 (250 필드)
4. **AI 강의 2026** — 시사·기술 도서 (650 인공지능)
5. **작별인사** — 같은 DDC 다른 fingerprint 비교용

## 알려진 한계

- 실제 ISBN 외부 API 호출, DDC 매칭 알고리즘, MARC 생성 알고리즘은 모두 시드 데이터로 대체.
- 5권 외 ISBN 입력 시 도서정보 호출 버튼 비활성.
- 승인은 메모리 상태만 변경 (페이지 새로고침 시 대시보드 카운트 27/40으로 복구).
- "전체 27건 보기" / "MARC 파일 내보내기" / 완료 리스트 "상세 →" 는 시연용 alert만 표시.
