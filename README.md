# Biblo: 학술정보관 MARC 자동 생성기 — 목업

시연용 정적 HTML 목업. `index.html`을 더블클릭하면 브라우저에서 바로 열립니다.

## 5단계 흐름
1. ISBN 입력 + 대시보드 + 완료 MARC 리스트
2. 도서 기본 정보 호출 + 책 지문 17필드
3. DDC 분류번호 자동 매칭
4. MARC 레코드 자동 생성
5. 최종 검수 (사람 책임 확인)

## 시연용 도서 5권
- 소년이 온다 / 한강
- 작별인사 / 김영하
- 사피엔스 / 유발 하라리
- 정재승의 과학콘서트 / 정재승
- 박태웅의 AI 강의 2026 / 박태웅

## 데이터 갈아끼우기
`data/*.js` 안의 `window.SAMPLE_BOOKS` 등 전역 객체를 직접 수정하면 됩니다.
